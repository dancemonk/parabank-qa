# TC-EXP-009 — Transfer to the same account

## Preconditions
- User is logged in
- User has at least one account

## Steps
1. Navigate to Transfer Funds
2. Select an account in the From field
3. Select the same account in the To field
4. Enter a valid amount
5. Click Transfer

## Expected Result
System displays an error or warning indicating source and destination accounts cannot be the same. Transfer does not proceed.

## Actual Result
System displays "Transfer Complete!"

## Priority
Medium

## Technique
Exploratory
